# 3D Reconstruction of Archaeological Artifacts

A Computer Vision and Image Processing project that reconstructs and visualizes 3D depth maps of archaeological artifacts from 2D images using Singular Value Decomposition (SVD) and interactive 3D visualization techniques.

This project demonstrates how Artificial Intelligence and image processing can help in the digital preservation and structural analysis of historical artifacts.

---

# 📌 Project Overview

Archaeological artifacts are often damaged, incomplete, or difficult to preserve physically. Digital reconstruction techniques provide an efficient way to analyze and visualize these artifacts without direct physical interaction.

This project transforms 2D images of archaeological objects into interactive 3D surface models using image processing and dimensionality reduction methods. The generated visualization helps in understanding the structure, texture, and depth characteristics of artifacts.

The project combines:

* Computer Vision
* Image Processing
* Matrix Decomposition
* Interactive Visualization
* Digital Heritage Preservation

# 🧠 Working Principle

## Step 1: Image Selection

The user selects an archaeological artifact image using a graphical file selection interface.

---

## Step 2: Image Processing

The selected image is processed to enhance important structural information and surface details.

---

## Step 3: Feature Extraction

Edge detection techniques identify contours and artifact boundaries to preserve significant geometric features.

---

## Step 4: Dimensionality Reduction

Singular Value Decomposition (SVD) is used to reduce complexity while retaining dominant structural patterns of the artifact.

---

## Step 5: Depth Reconstruction

The processed image data is transformed into a depth representation that simulates a 3D surface structure.

---

## Step 6: Interactive 3D Visualization

The reconstructed surface is displayed as an interactive 3D mesh model that can be rotated, zoomed, and explored in the browser.
